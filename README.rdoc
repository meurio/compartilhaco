Querido voluntário,

Nas próximas semanas, o Multitude será desativado e queríamos explicar para você o motivo dessa decisão. Primeiro, fique tranquilo: vamos continuar oferecendo oportunidades de ação de para nossos membros. No entanto, o Multitude como você conhece vai acabar, e o motivo é bem simples: desenvolvemos um jeito mais rápido e prático de conectar os cidadãos com as causas de seu interesse. Agora, você só precisa preencher seu perfil na rede, que nossa equipe vai selecionar e enviar as melhores oportunidades de ação diretamente pra você, por email! Simples, não é?

Caso tenha alguma dúvida ou sugestão, mande um email para {contato@nossascidades.org}[mailto:contato@nossascidades.org] ou visite nosso site: {Nossas Cidades}[http://www.nossascidades.org].

== Setup
=== DISCLAIMER: This project is no longer maintained.

1. Install Ruby on Rails
2. Edit config/database.yml
3. Run bundle exec rake db:create db:migrate db:test:prepare
4. Run rails s and access http://localhost:3000
