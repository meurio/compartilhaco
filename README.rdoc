Querido membro,

Nas próximas semanas, o Compartilhaço será desativado. Nós poderíamos escrever um longo texto para introduzir essa notícia, mas não gostamos de enrolação ;). Nos últimos meses, após uma série de mudanças nos algoritmos das redes sociais, percebemos que esta ferramenta não estava mais garantindo o impacto e a visibilidade prometidas às postagens. Diante desse cenário entendemos que não fazia mais sentido manter o Compartilhaço no ar.

Para nós é muito difícil dar adeus a uma ferramenta tão bacana quanto esta, que ajudou mais de 150 mobilizações a ampliarem seu impacto em quase 2 anos. No entanto, ficamos tranquilos em fazer isso acompanhado de duas boas notícias:

Neste momento, nossa equipe já está desenvolvendo novas tecnologias de mobilização social de ponta, que prometem causar muito rebuliço. Estamos muito empolgados com isso, e temos certeza que você também vai adorar as novidades :)
Existe uma ferramenta semelhante ao compartilhaço, o {Thunderclap}[https://www.thunderclap.it/], que tem uma versão em português. Ou seja, você vai continuar podendo sincronizar suas postagens!

Por enquanto é isso, mas como sempre, estamos à disposição para conversar quando quiser. É só mandar um email para {contato@nossascidades.org}[mailto:contato@nossascidades.org] ou visite nosso site: {Nossas Cidades}[http://www.nossascidades.org].


== Setup
=== DISCLAIMER: This project is no longer maintained.

1. Install Ruby on Rails
2. Edit config/database.yml
3. Run bundle exec rake db:create db:migrate db:test:prepare
4. Run rails s and access http://localhost:3000
